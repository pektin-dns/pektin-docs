# Record Types

# Simple Records

## AAAA

```json
{
    "name": "pektin.xyz.:AAAA",
    "rr_set": [
        {
            "ttl": 3600,
            "value": {
                "AAAA": "1::1"
            }
        }
    ]
}
```

# Complex Records

Called complex because they have extra fields instead of one string.

## MX

```json
{
    "name": "pektin.xyz.:MX",
    "rr_set": [
        {
            "ttl": 3600,
            "value": {
                "MX": {
                    "preference": 10,
                    "exchange": "mx.pektin.xyz."
                }
            }
        }
    ]
}
```

## SRV

```json
{
    "name": "pektin.xyz.:SRV",
    "rr_set": [
        {
            "ttl": 3600,
            "value": {
                "SRV": {
                    "priority": 1,
                    "weight": 1,
                    "port": 443,
                    "target": "mx.pektin.xyz."
                }
            }
        }
    ]
}
```
